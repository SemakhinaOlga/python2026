# Практическое задание 35.1

Это задание относится к первому блоку темы "Функции". Вы уже знакомы с коллекциями и алгоритмами. Сейчас вы начнёте изучать основы функций: как их объявлять, передавать в них данные и получать результат. Это фундаментальный навык, который позволит вам структурировать код и избегать повторений. Данное упражнение подготовит вас к работе с более сложными аспектами функций, таким как значения по умолчанию и именованные аргументы.

## Задания

### Описание задачи
Ольга, вы помогаете разработать модуль для простого калькулятора площадей. Ваша задача — создать функцию, которая вычисляет площадь прямоугольника. Функция должна принимать два числа (ширину и высоту) в качестве входных данных и возвращать вычисленную площадь. Не печатайте результат внутри функции, только возвращайте его.

**Контекст:** В программе уже есть код, который будет использовать вашу функцию для расчёта материалов (например, сколько краски нужно для стены). Поэтому важно, чтобы функция именно *возвращала* число, а не печатала его.

### Пример входных и выходных данных
**Пример 1:**
*   **Входные данные для функции:** `width = 5`, `height = 3`
*   **Что возвращает функция:** `15`
*   **Как может использоваться в программе:**
    ```python
    # Ваша функция будет вызываться так:
    wall_area = calculate_area(5, 3)
    # Переменная wall_area теперь содержит число 15
    paint_needed = wall_area * 0.5  # Предположим, 0.5 литра на кв.м.
    print(f"Требуется {paint_needed} литров краски.")  # Выведет: Требуется 7.5 литров краски.
    ```

**Пример 2:**
*   **Входные данные для функции:** `width = 10`, `height = 2`
*   **Что возвращает функция:** `20`
*   **Пояснение:** Функция получает числа 10 и 2, перемножает их и возвращает результат 20. Этот результат можно сохранить в переменную или использовать сразу в выражении, например `print(calculate_area(10, 2) * 2)` выведет `40`.

### Критерии проверки и ограничения
1.  Функция должна быть объявлена с помощью ключевого слова `def`.
2.  Функция должна иметь два параметра (например, `width` и `height`).
3.  Внутри функции должно происходить вычисление площади (ширина * высота).
4.  Функция должна использовать оператор `return` для возврата результата вычисления.
5.  **Запрещено** использовать `print()` внутри функции для вывода площади.
6.  Не используйте значения по умолчанию, `*args`, `**kwargs` или именованные аргументы при вызове — передавайте аргументы позиционно.

### Решение задачи (псевдокод)
1.  Объявить функцию с именем (например, `calculate_area`), которая принимает два параметра.
2.  Внутри тела функции:
    a.  Создать переменную `result` (или использовать сразу выражение).
    b.  Присвоить этой переменной значение произведения первого параметра на второй.
3.  Вернуть значение переменной `result` (или результат вычисления) из функции с помощью соответствующего оператора.

### Рекомендации
*   **Вынести в отдельную функцию:** Всю логику вычисления площади необходимо поместить в функцию `calculate_area`. Это и есть цель задания.
*   **PEP8:**
    *   Используйте осмысленные имена для функции и её параметров (например, `calculate_area`, `width`, `height`).
    *   После объявления функции `def ...:` оставьте две пустые строки перед следующим блоком кода.
