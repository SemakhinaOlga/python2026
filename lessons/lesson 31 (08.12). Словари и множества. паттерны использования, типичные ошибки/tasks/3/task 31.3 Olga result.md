### Отчёт по проверке задания

---

**1. Формулировка задания**  
Требуется сравнить два множества уязвимостей (веб и мобильное приложение), вывести:
- Общие уязвимости (пересечение множеств)
- Уникальные уязвимости только в мобильном приложении (разность множеств)

**Ограничения:**
- Использовать операции над множествами (`&`, `-`, `|`)
- Не преобразовывать множества в списки без необходимости
- Вывод должен содержать пояснительные подписи

**Пример ввода/вывода:**
```python
# Ввод
set_a = {'IDOR', 'CSRF', 'XSS'}
set_b = {'SSRF', 'MITM', 'CSRF'}

# Вывод
Общие уязвимости: {'CSRF'}
Только в мобайле: {'SSRF', 'MITM'}
```

---

**2. Результаты проверки**  
**Файл:** `task 3.py`  
**Код:**
```python
set_a = {'IDOR', 'CSRF', 'XSS'} #веб
set_b = {'SSRF', 'MITM', 'CSRF'} #мобайл
print('Общие уязвимости: ', set_a & set_b)
print('Только в мобайле: ', set_b-set_a)
```

**Тестирование:**
- Запущен код студента → вывод совпадает с примером (порядок элементов не важен).
- Проверены операции:
  - `set_a & set_b` → корректное пересечение `{'CSRF'}`
  - `set_b - set_a` → корректная разность `{'SSRF', 'MITM'}`
