## Задача 1. Учёт часов команд в коворкинге

Пример входных данных:
```python
bookings = [
    ("designers", 8),
    ("marketing", 5),
    ("designers", 4),
    ("startups", 6),
]
finished = ["marketing"]
```

### Требуется
1. Создать пустой словарь `hours`.
2. В цикле по `bookings` накапливать часы по каждой команде через `hours[team] = hours.get(team, 0) + time`.
3. Для каждой команды из `finished` удалить запись из **того же** словаря `hours` через `pop(team, "0 часов")`.
4. Вывести понятное сообщение об удалении и итоговый словарь.

### Критерии
- Дубликаты ключей не теряются, а суммируются.
- Удаление происходит из основного словаря, а не из вспомогательной структуры.
