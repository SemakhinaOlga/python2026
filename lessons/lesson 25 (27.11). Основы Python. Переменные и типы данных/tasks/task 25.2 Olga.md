Задание 2. Основы Python. Переменные и типы данных

### Описание
Сделай программу, которая считывает название курса, длительность в неделях и стоимость одного занятия. Приведи длительность к int, стоимость к float. Рассчитай общую стоимость и выведи все значения с подписями и типами.

### Пример входных данных
```
Python for QA
12
850.0
```

### Пример выходных данных
```
Курс: Python for QA (str)
Длительность: 12 (int)
Стоимость занятия: 850.0 (float)
Полная стоимость: 10200.0 (float)
```

### Критерии успешного выполнения
- Данные читаются через input() в указанном порядке.
- Длительность преобразуется к int, стоимость к float.
- Итоговая стоимость рассчитывается как произведение.
- Вывод содержит подписи, значения и типы.
- Скрипт завершается без ошибок при корректном вводе.

### Подсказки
- Сохрани результат type(значение).__name__ в переменной, если используется несколько раз.
- Применяй f-строки для компактного вывода.
- Проверь порядок преобразований, чтобы не получить ValueError.

### Частые ошибки и ограничения
- Не меняй порядок чтения значений.
- Не округляй результат вручную.
- Не добавляй сторонние библиотеки или ввод дополнительных данных.
