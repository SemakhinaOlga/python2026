### Блок 1: Работа со списками

Ты ведешь список ингредиентов для торта. Создай список `base_ingredients` с тремя ингредиентами. Добавь два новых ингредиента в конец списка. Выведи общее количество ингредиентов, первый и последний ингредиент. Затем выведи нумерованный список всех ингредиентов.

Пример входных данных:
```python
base_ingredients = ["мука", "сахар", "яйца"]
additional_ingredients = ["масло", "разрыхлитель"]
```

### Блок 2: Срезы (особое внимание)

У тебя есть список времени приготовления блюд (в минутах). 
1. Выведи первые четыре времени с подписью "Быстрые блюда".
2. Выведи последние три времени с подписью "Долгие блюда".
3. Выведи срез со второго по пятый элемент включительно с подписью "Среднее время".
4. Выведи каждое третье время, начиная с первого, с подписью "Каждое третье блюдо".

Пример входных данных:
```python
cooking_times = [10, 15, 20, 25, 30, 35, 40, 45, 50]
```

### Блок 3: Мутация списков

У тебя есть список шагов по уборке квартиры. 
1. Добавь в конец списка шаги: "протереть пыль" и "пропылесосить".
2. Вставь в середину списка (после второго элемента) шаг "помыть окна".
3. Удали из списка шаг "вынести мусор" (если он есть).
4. Перемести последний шаг в начало списка.

Пример входных данных:
```python
steps = ["полить цветы", "вынести мусор", "помыть посуду", "протереть полки"]
```

### Блок 4: Вложенные структуры

У тебя есть список рецептов, каждый рецепт представлен словарем с полями: `название`, `тип` (десерт/основное/закуска), `время_приготовления` (в минутах).

3. Создай новый список только с десертами (тип "десерт").

    {"название": "Салат Цезарь", "тип": "закуска", "время_приготовления": 20},
