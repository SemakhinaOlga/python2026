# Задание 29.2. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Индексация и срезы. Учимся брать подсписки с разными границами и шагом, чтобы анализировать списки без дополнительного кода.

## Условие задачи
Ты ведешь список длительности глав аудиокниги. Нужно:
1. Получить первые четыре главы для трейлера.
2. Показать последние три главы, чтобы оценить концовку.
3. Сформировать срез, который возвращает главы со второй по шестую включительно.
4. Собрать каждую вторую главу, начиная с первой, чтобы составить укороченную версию.

## Пример входных данных
```python
durations = [12, 15, 14, 17, 19, 18, 16, 20, 22]
```

## Пример выходных данных
```
Начало книги: [12, 15, 14, 17]
Финал: [18, 16, 20, 22]
Средний блок: [15, 14, 17, 19, 18]
Каждая вторая: [12, 14, 19, 16, 22]
```

## Критерии успешного выполнения
- Все операции реализованы через синтаксис срезов
- Используются как положительные, так и отрицательные индексы
- Каждый срез снабжен пояснением
- Исходный список не изменяется

## Подсказки
- Пустой `start` удобен для выбора начала списка
- Чтобы включить шестой элемент, поставь `stop=7`
- Отрицательный `start` помогает захватить хвост
- Шаг `2` выбирает через один

## Частые ошибки и ограничения
- Не ставь шаг 0 — это ошибка
- Не путай границы: `stop` всегда исключается
- Следи, чтобы итоговые срезы не выходили за длину списка
- Не пытайся комбинировать срезы с `append`, вся работа через `[:]`
