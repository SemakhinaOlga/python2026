# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Учимся уверенно добавлять, вставлять и удалять элементы, сохраняя контроль над порядком задач.

## Условие задачи
Ты организуешь backstage для школьного спектакля:
1. Создай список ответственных за участки сцены.
2. Добавь в конец список волонтеров, присланный куратором, через `extend`.
3. Вставь специалиста по свету в позицию №2, чтобы он шёл сразу после ведущего.
4. Удали по значению человека, который заболел.
5. Перемести первый элемент списка в конец, используя `pop(0)` и `append`.
6. Выведи итоговый порядок с подписью роли и имени.

## Пример входных данных
```python
crew = ["ведущий, Антон", "звук, Ира", "декорации, Лена"]
volunteers = ["костюмы, Вика", "логистика, Рома"]
```

## Пример выходных данных
```
1. ведущий, Антон
2. свет, Гульнара
3. звук, Ира
4. декорации, Лена
5. костюмы, Вика
```

## Критерии успешного выполнения
- Применены минимум три разных метода мутации
- Вставка и удаление выполняются по корректным индексам/значениям
- Перемещение реализовано методами списка без ручного обмена переменных
- Итоговый список напечатан после всех операций

## Подсказки
- `extend` добавит всех волонтеров одним вызовом
- Перед `remove` убедись, что нужная роль реально присутствует
- `pop(0)` возвращает первый элемент. его можно сразу передать в `append`

## Частые ошибки и ограничения
- Не модифицируй список в том же цикле, где его перебираешь
- Не вызывай `remove`, если не уверен в наличии элемента
- Следи, чтобы индексы вставки были внутри диапазона
- Не создавай новый список вместо работы с первоначальным `crew`
