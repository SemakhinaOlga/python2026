# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Важно понять, как хранить набор данных в списке, быстро получать доступ к элементам и аккуратно их перебирать.

## Условие задачи
Подготовь список книг для зимнего марафона чтения. Создай базовый список, добавь рекомендации подруги в конец, выведи общее количество книг, покажи первую и предпоследнюю позиции. Затем распечатай нумерованный список, чтобы отметить порядок чтения.

## Пример входных данных
```python
winter_books = ["Атлант", "Три товарища", "Скандинавские боги"]
friends_choice = ["Дом в котором", "Унесенные ветром"]
```

## Пример выходных данных
```
Всего книг: 5
Первая книга: Атлант
Предпоследняя книга: Дом в котором
1. Атлант
2. Три товарища
3. Скандинавские боги
4. Дом в котором
5. Унесенные ветром
```

## Критерии успешного выполнения
- Исходный список создается в коде и расширяется новыми элементами
- Используется `len` для подсчета количества книг
- Доступ к первой и предпоследней книге реализован индексами
- Итоговый список выводится в отдельные строки с номерами

## Подсказки
- Чтобы получить предпоследний элемент, воспользуйся индексом `-2`
- Метод `extend` быстрее добавит несколько книг сразу
- Вывод с `enumerate` избавит от ручного счетчика

## Частые ошибки и ограничения
- Не обращайся к индексу `-2`, если длина списка меньше двух
- Избегай повторного создания списка через конкатенацию строк
- Следи, чтобы вывод происходил уже после добавления новых книг
- Не смешивай типы данных, элементы списка должны оставаться строками
