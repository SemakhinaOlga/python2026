### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая формирует аккуратный чек по списку покупок.

Требования:
- прочитать строку с покупками формата `название: цена` через запятую
- разобрать строку в список пар (название, цена)
- удалить пробелы вокруг названия, привести его к нижнему регистру
- посчитать итоговую сумму покупок
- вывести список покупок построчно и общую сумму

### Пример входных данных
```
  Хлеб: 40, Молоко: 70, сыр: 120 
```

### Пример выходных данных
```
хлеб . 40
молоко . 70
сыр . 120
Итого . 230
```

### Критерии успешного выполнения
- строка корректно разбирается по запятым и двоеточию
- названия очищены и нормализованы
- суммы посчитаны верно

### Подсказки
- используй `split(',')` для покупок и `split(':')` для пары
- не забудь обрабатывать пробелы с помощью `strip()`

### Частые ошибки и ограничения
- обработай ситуацию некорректного формата пары
- не используй внешние библиотеки
