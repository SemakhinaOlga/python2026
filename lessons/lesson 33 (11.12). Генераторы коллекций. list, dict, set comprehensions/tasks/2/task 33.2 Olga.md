# Задание 33.2. Генераторы коллекций. list, dict, set comprehensions

## Блок урока
«List comprehensions» помогает аккуратно сочетать преобразование и фильтрацию списков без лишних переменных.

## Условие задачи
**Сложность**: <span style="color: #f9a825">● средний</span>. примерное время: 15–20 минут

У тебя есть список словарей `orders`, где каждый словарь содержит ключи `"id"`, `"items"` (количество позиций) и `"status"`. Построй список `pending_orders`, который содержит строки формата `"Заказ #ID: ITEMS позиций"` только для заказов со статусом `"pending"` и количеством позиций от 1 до 5 включительно. Используй одно выражение list comprehension.

## Пример входных данных
```
orders = [
    {"id": 110, "items": 2, "status": "pending"},
    {"id": 111, "items": 6, "status": "pending"},
    {"id": 112, "items": 4, "status": "done"},
    {"id": 113, "items": 1, "status": "pending"}
]
```

## Пример выходных данных
```
['Заказ #110: 2 позиций', 'Заказ #113: 1 позиций']
```

## Критерии успешного выполнения
- Результат формируется единственным list comprehension.
- В итог попадают только заказы со статусом `pending` и количеством 1–5.
- Формат строки полностью совпадает с образцом.
- Исходный список не изменяется.

## Подсказки
- Перебирай элементы как `for order in orders`.
- Фильтр записывается в конце comprehension через `if`.
- Строку удобно собрать через f-строку с подстановкой значений.

## Частые ошибки и ограничения
- Не добавляй отдельные циклы или `append`.
- Не забывай про верхнюю границу 5.
- Следи, чтобы списки не имели больше пяти пунктов.
